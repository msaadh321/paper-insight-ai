# Paper Insight AI 🧠📄

[![Python](https://img.shields.io/badge/Python‑3.11‑blue?logo=python&logoColor=white)](https://www.python.org/)
[![Streamlit](https://img.shields.io/badge/Streamlit‑1.26‑orange?logo=streamlit&logoColor=white)](https://streamlit.io/)
[![License](https://img.shields.io/badge/License‑MIT‑green)](LICENSE)

**Paper Insight AI** is a modern academic research paper analyzer — an AI‑powered tool built to help students, researchers, and academics quickly extract insights, summaries, and important sections from research papers.

This project uses **Python, spaCy, BERT, and Gemini API** combined with a clean **Streamlit** web interface for fast, intelligent analysis.

---

## 🚀 Features

✔ Upload and preview research papers (PDF/Text)  
✔ Generate concise and detailed summaries  
✔ Extract key sections like Abstract, Methodology, Results & Conclusion  
✔ NLP‑powered insights using spaCy & BERT  
✔ Ask questions about the paper — AI‑powered Q&A  
✔ Keyword extraction & topic analysis  
✔ Download results or export analysis reports

---

## 📌 How It Works

1. Upload an academic research paper (PDF or text).  
2. Extract text and preprocess using **spaCy** & custom logic.  
3. Use **BERT** for summarization and **Gemini API** for enhanced AI responses.  
4. Display key sections, insights, and charts in **Streamlit**.  
5. Allow downloads or sharing of summary reports.
